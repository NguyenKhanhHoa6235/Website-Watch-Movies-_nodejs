Database:
Vì nhóm thực hiện đề tài bằng MongoDB compass, qua nhiều lần tìm hiểu thì nhóm không biết cách để export
một database gồm nhiều collections như thế nào, chính vì thế nhóm đã export từng collections.

Khi thực hiện chạy, import từng file json với từng collections phù hợp.

users -> import users.json
movies -> import movies.json
categories -> import categories.json
